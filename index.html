ECサイト開発プロジェクト 紹介

Spring BootとThymeleafを活用した実践的なECサイト構築

主な機能

ECサイトとして必要な機能を一通り実装。学習・参照・拡張のベースとして活用できます。

  • 会員管理: 会員登録・ログイン・退会機能
  • 商品閲覧: 商品一覧・カテゴリ分類・商品検索
  • ショッピング: カート（買い物かご）への商品追加・数量変更・削除
  • 注文管理: 注文処理・注文履歴の参照
  • 管理者機能: 管理者による商品・カテゴリ・会員のCRUD管理

技術スタック

Spring エコシステムをベースに、フロントエンドは Thymeleaf を採用。

Java 17 Spring Boot 4 Spring Data JPA (Hibernate) Thymeleaf HTML5 / CSS3 Oracle Database

設計のポイント

標準的なMVC構造に加え、独自の工夫が随所に施されています。

  • EntityとBeanの分離: データベースモデル(Entity)とビューモデル(Bean)を分離し、BeanToolsで柔軟に変換。
  • 価格計算の切り出し: 複雑化しやすい価格・税金計算をPriceCalcクラスに集約し、保守性を向上。
  • 画像アップロード管理: UploadFileServiceにより、商品画像の安全なアップロードと紐付けを実現。
