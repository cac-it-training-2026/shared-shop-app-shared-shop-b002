主な機能

標準的なECサイトの基本機能に加え、より実践的な拡張機能も実装されています。

👥 会員管理

会員登録・ログイン・退会機能に加え、不正アクセスを防ぐアカウントロック機能を実装。

🔍 商品閲覧

商品一覧・カテゴリ分類・検索機能と、ユーザーからのフィードバックを集める商品レビュー機能。

🛒 ショッピング

カートへの商品追加・数量変更・削除、および気になる商品を保存できるお気に入り登録機能。

📦 注文管理

注文処理・注文履歴の参照に加え、購入時に割引を適用できるクーポン機能。

⚙️ 管理者機能

管理者による商品・カテゴリ・会員情報のCRUD操作を含む一元管理機能。

画面一覧

各機能の実際の画面構成です。

ログイン画面

ログインするユーザーによって表示画面が変わります。

商品一覧

トップページおよびカテゴリごとの商品を表示します。

ショッピングカート

選択した商品の確認と合計金額を計算します。

注文確認

お届け先やお支払方法の最後の確認を行います。

管理者ダッシュボード

会員や商品の情報を一元管理します。

技術スタック

Spring エコシステムをベースに、フロントエンドは Thymeleaf を採用。

Java 17 Spring Boot 4 Spring Data JPA Hibernate Thymeleaf HTML5 / CSS3 Oracle Database

設計のポイント

標準的なMVC構造に加え、独自の工夫が随所に施されています。

EntityとBeanの分離

データベースモデル(Entity)とビューモデル(Bean)を分離し、BeanToolsで柔軟に変換。

価格計算の切り出し

複雑化しやすい価格・税金計算をPriceCalcクラスに集約し、保守性を向上。

画像アップロード管理

UploadFileServiceにより、商品画像の安全なアップロードと紐付けを実現。
