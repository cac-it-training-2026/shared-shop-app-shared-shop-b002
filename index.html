ECサイト開発プロジェクト 紹介

Spring BootとThymeleafを活用した実践的なECサイト構築

主な機能

ECサイトとして必要な機能を一通り実装。学習・参照・拡張のベースとして活用できます。

  • 会員登録・ログイン・退会機能
  • 商品一覧・カテゴリ分類・商品検索
  • カート（買い物かご）への商品追加・削除
  • 注文処理・注文履歴の参照
  • 管理者による商品・カテゴリ・会員のCRUD管理

技術スタック

Spring エコシステムをベースに、フロントエンドは Thymeleaf を採用。

  • バックエンド: Java 17, Spring Boot 4
  • フロントエンド: HTML5, CSS3, Thymeleaf
  • データベース: Oracle Database
  • O/Rマッパー: Spring Data JPA (Hibernate)

設計のポイント

標準的なMVC構造に加え、独自の工夫が随所に施されています。

  • EntityとBeanの分離: データベースモデルとビューモデルを分離し、BeanToolsで柔軟に変換。
  • 価格計算の切り出し: 複雑化しやすい価格・税金計算をPriceCalcクラスに集約。
  • 画像アップロード管理: UploadFileServiceにより、商品画像の安全なアップロードと紐付けを実現。
